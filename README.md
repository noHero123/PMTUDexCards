# PMTUDex Cards
All cards with QR code that [PMTUDex](https://github.com/noHero123/PMTUDex) can handle

you need:
 - everything from [BasicItems](BasicItems) (only one genXtm.pdf is needed)
 - in  [Other folder](Other) you need 1x [field_moves.pdf](Other/field_moves.pdf) and 6x [status_cards.pdf](Other/status_cards.pdf) (one per player)
 - a trainer pdf of your choosen generation from [Trainer folder](Trainer)

additional you need the non-supported cards from PMTU:
in [PMTU Drive](https://drive.google.com/drive/folders/1UshNmrg7SGDPV_wEpOVCtRzFja5hPo3X) 
you need:
 - Shop Items.pdf *
 - Master Balls.pdf
 - Event Cards.pdf *
 - Map Specific Items.pdf
 - Poké Balls.pdf
 - Base Items.pdf *

* replace the cards of the orginal pdfs with the QR-cards.
